# LiteLLM-CLIProxyAPI
A self-hosted LLM proxy service combining [LiteLLM](https://github.com/BerriAI/litellm) and [CLIProxyAPI](https://github.com/router-for-me/CLIProxyAPI) to enable integration with a wider variety of LLM providers. Includes monitoring capabilities and is deployed on Docker Swarm, managed via Portainer.
